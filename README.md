<h1 align="left">Calculator</h1>

<p align="left">-This project is a simple calculator
using JavaScript, HTML, and CSS. All operations are done
without built in functions and instead use
custom functions.
</p>

## Features

- [x] Multiply
- [x] Divide
- [x] Add
- [x] Subtract
- [x] Clear
- [x] Backspace
